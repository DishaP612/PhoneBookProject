# Phone Book Management System

## Project Description
This project is a Phone Book Management System developed using Python, Dictionary, and Tkinter GUI.  
The application allows users to store, search, update, delete, and display contacts.

## Features
- Add Contact
- Search Contact
- Update Contact
- Delete Contact
- Display All Contacts
- Save Contacts to File
- GUI Interface using Tkinter

## Technologies Used
- Python
- Tkinter
- Dictionary Data Structure
- File Handling
- GitHub

## How to Run the Project
1. Open project in PyCharm
2. Run main.py
3. Phone Book GUI window will open
